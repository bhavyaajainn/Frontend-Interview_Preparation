# Program to Check Prime Number

---

## Problem Statement

Given a number, check if it is a prime number. A prime number is a number that is divisible by only 1 and itself.

---

## Examples

### Example 1

**Input:**

```
1
```

**Output:**

```
false
```

**Explanation:**
1 is not a prime number.

---

### Example 2

**Input:**

```

```

**Output:**

```
true
```

**Explanation:**
2 is a prime number.

## Constraints

- 0 ≤ n ≤ 10^6

---

## Time Complexity

- O(√n) for both iterative and recursive solutions.

---

## Space Complexity

- O(1) for iterative, O(√n) for recursive (call stack).
