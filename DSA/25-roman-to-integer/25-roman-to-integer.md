# Convert Roman Numeral to Integer

---

## Problem Statement

Given a Roman numeral as a string, convert it to its integer value.

---

## Examples

### Example 1

**Input:**

```
IV
```

**Output:**

```
4
```

**Explanation:**
I (1) before V (5) means 5 - 1 = 4.

---

### Example 2

**Input:**

```
XXIX
```

**Output:**

```
29
```

**Explanation:**
XX (20) + IX (9) = 29.

---

### Example 3

**Input:**

```
XVII
```

**Output:**

```
17
```

**Explanation:**
X (10) + V (5) + II (2) = 17.

---

### Example 4

**Input:**

```
XIV
```

**Output:**

```
14
```

**Explanation:**
X (10) + IV (4) = 14.

---

## Constraints

- 1 ≤ length of roman ≤ 15
- Roman numeral is valid and uses uppercase letters

---

## Time Complexity

- O(n), where n is the length of the string

---

## Space Complexity

- O(1)
