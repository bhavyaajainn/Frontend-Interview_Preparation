# Find Factorial of a Number

---

## Problem Statement

Given a number, find its factorial using three different methods: brute force, recursion, and stack.

---

## Examples

### Example 1

**Input:**

```
5
```

**Output:**

```
120
```

**Explanation:**
5! = 5 × 4 × 3 × 2 × 1 = 120

---

### Example 2

**Input:**

```
10
```

**Output:**

```
3628800
```

**Explanation:**
10! = 3628800

---

### Example 3

**Input:**

```
12
```

**Output:**

```
479001600
```

**Explanation:**
12! = 479001600

---

### Example 4

**Input:**

```
0
```

**Output:**

```
1
```

**Explanation:**
0! = 1 by definition.

---

## Constraints

- 0 ≤ num ≤ 20 (for safe integer results in JavaScript)

---

## Time Complexity

- O(n) for all methods

---

## Space Complexity

- O(1) for brute force, O(n) for recursion and stack
