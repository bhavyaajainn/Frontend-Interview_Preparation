# Converting String to JadenCase

---

## Problem Statement

Given a string, convert it to JadenCase, where every first letter of each word is uppercase.

---

## Examples

### Example 1

**Input:**

```
How can mirrors be real if our eyes aren't real
```

**Output:**

```
How Can Mirrors Be Real If Our Eyes Aren't Real
```

**Explanation:**
Each word's first letter is capitalized.

---

### Example 2

**Input:**

```
hello world
```

**Output:**

```
Hello World
```

**Explanation:**
Each word's first letter is capitalized.

---

### Example 3

**Input:**

```
a b c
```

**Output:**

```
A B C
```

**Explanation:**
Each word's first letter is capitalized.

---

### Example 4

**Input:**

```

```

**Output:**

```

```

**Explanation:**
Empty string returns empty string.

---

## Constraints

- 0 ≤ length of string ≤ 10⁵
- String may contain any characters

---

## Time Complexity

- O(n), where n is the length of the string.

---

## Space Complexity

- O(n), for storing the result string.
