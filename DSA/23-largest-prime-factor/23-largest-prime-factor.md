# Find the Largest Prime Factor

---

## Problem Statement

Given a number, find its largest prime factor.

---

## Examples

### Example 1

**Input:**

```
600851475143
```

**Output:**

```
6857
```

**Explanation:**
6857 is the largest prime factor of 600851475143.

---

### Example 2

**Input:**

```
151
```

**Output:**

```
151
```

**Explanation:**
151 is a prime number, so it is its own largest prime factor.

---

### Example 3

**Input:**

```
256987513645261
```

**Output:**

```
36712501949323
```

**Explanation:**
36712501949323 is the largest prime factor.

---

### Example 4

**Input:**

```
231412151232312
```

**Output:**

```
9642172968013
```

**Explanation:**
9642172968013 is the largest prime factor.

---

## Constraints

- 2 ≤ num ≤ 10^15

---

## Time Complexity

- O(√n)

---

## Space Complexity

- O(1)
