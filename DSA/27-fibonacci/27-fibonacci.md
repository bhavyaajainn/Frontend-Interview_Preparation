# Program to Find the nth Fibonacci Number

---

## Problem Statement

Given a number n, find the nth Fibonacci number using three different approaches: iterative, recursion, and dynamic programming (memoization).

---

## Examples

### Example 1

**Input:**

```
10
```

**Output:**

```
55
```

**Explanation:**
The 10th Fibonacci number is 55.

---

### Example 2

**Input:**

```
12
```

**Output:**

```
144
```

**Explanation:**
The 12th Fibonacci number is 144.

---

### Example 3

**Input:**

```
9
```

**Output:**

```
34
```

**Explanation:**
The 9th Fibonacci number is 34.

---

### Example 4

**Input:**

```
0
```

**Output:**

```
0
```

**Explanation:**
The 0th Fibonacci number is 0.

---

## Constraints

- 0 ≤ n ≤ 50 (for safe integer results in JavaScript)

---

## Time Complexity

- Iterative: O(n)
- Recursive: O(2^n)
- Dynamic Programming: O(n)

---

## Space Complexity

- Iterative: O(1)
- Recursive: O(n) (call stack)
- Dynamic Programming: O(n)
