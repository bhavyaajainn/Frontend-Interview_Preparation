# Two Sum

## Problem Statement

Given an unsorted array of integers, determine whether there exists a pair whose sum equals a given number `k`.

Return `true` if such a pair exists, otherwise return `false`.

---

## Examples

### Example 1

Input:
arr = [15, 4, 9, 3, 2, 12, 11, 14, 21, 24, 1, 10]
k = 25

Output:
true

Explanation:
Possible pairs → (15 + 10), (4 + 21), (11 + 14), (24 + 1)

---

### Example 2

Input:
arr = [1, 2, 3, 4, 5]
k = 9

Output:
true

Explanation:
(4 + 5) = 9

---

### Example 3

Input:
arr = [1, 2, 3, 4, 5]
k = 10

Output:
false

Explanation:
No pair sums to 10.

---

### Example 4

Input:
arr = [5, -2, 8, 1]
k = 6

Output:
true

Explanation:
(5 + 1) = 6

---

## Constraints

- The array is unsorted.
- Return only a boolean value.
- You do not need to return the actual pair.
- Focus only on checking existence of such a pair.

---

## Time Complexity

- Brute Force: O(n²)
- Sorting + Two Pointers: O(n log n)
- Hashing (Optimal): O(n)

---

## Space Complexity

- Brute Force: O(1)
- Sorting + Two Pointers: O(1)
- Hashing (Optimal): O(n)
