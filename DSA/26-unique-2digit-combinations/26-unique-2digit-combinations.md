# Print All Unique 2 Digit Combinations of Given Numbers

---

## Problem Statement

Given an array of numbers, print all unique 2 digit combinations (pairs) in ascending order, with no duplicates.

---

## Examples

### Example 1

**Input:**

```
[1, 2, 3]
```

**Output:**

```
1 2
1 3
2 3
```

**Explanation:**
All unique pairs are printed.

---

### Example 2

```

```

**Output:**

```
1 2
1 3
1 4
1 5
2 3
2 4
2 5
3 4
3 5
4 5
```

**Explanation:**
All unique pairs are printed.

---

### Example 3

**Input:**

```

```

```

```

No unique pairs.

3 5

Pairs are printed in ascending order.

---

## Constraints

- 1 ≤ length of array ≤ 10⁴
- Array elements are integers

---

## Time Complexity

- O(n²), where n is the number of unique elements

---

## Space Complexity

- O(n), for storing unique elements
