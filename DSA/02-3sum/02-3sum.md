# 3 Sum Problem

---

## Problem Statement

Given an array of integers and a target sum, determine whether there exists a triplet in the array such that the sum of the three numbers is equal to the given target.

Return `true` if such a triplet exists, otherwise return `false`.

---

## Examples

### Example 1

**Input:**  
arr = [1, 2, 3, 5, 6, 11, 15, 16, 17, 18]  
target = 20

**Output:**  
true

**Explanation:**  
One valid triplet is (1, 3, 16) whose sum equals 20.

---

### Example 2

**Input:**  
arr = [4, 7, 1, 10, 2]  
target = 13

**Output:**  
true

**Explanation:**  
Triplet (4, 7, 2) sums up to 13.

---

### Example 3

**Input:**  
arr = [5, 2, 8, 14]  
target = 10

**Output:**  
false

**Explanation:**  
No combination of three numbers produces the target sum.

---

### Example 4

**Input:**  
arr = [2, 4, 6, 8, 10]  
target = 18

**Output:**  
true

**Explanation:**  
Triplet (2, 6, 10) sums up to 18.

---

## Constraints

- 3 ≤ n ≤ 10^5
- -10^9 ≤ arr[i] ≤ 10^9
- -10^9 ≤ target ≤ 10^9
- Array elements may contain duplicates.
- Triplets must use distinct indices.

---

## Time Complexity

O(n^2)

---

## Space Complexity

O(1)
