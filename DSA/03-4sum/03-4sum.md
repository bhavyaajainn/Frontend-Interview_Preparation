# 4 Sum Problem

---

## Problem Statement

Given an array of integers and a target sum, determine whether there exists a quadruplet (four elements) in the array such that their sum is equal to the given target.

Return `true` if such a quadruplet exists, otherwise return `false`.

Optionally, the problem may require printing all unique quadruplets whose sum equals the target.

---

## Examples

### Example 1

**Input:**  
arr = [1, 2, 3, 5, 6, 11, 15, 16, 17, 18]  
target = 20

**Output:**  
true

**Explanation:**  
One valid quadruplet is (1, 3, 5, 11) whose sum equals 20.

---

### Example 2

**Input:**  
arr = [4, 7, 1, 10, 2, 6]  
target = 19

**Output:**  
true

**Explanation:**  
Quadruplet (4, 7, 2, 6) sums up to 19.

---

### Example 3

**Input:**  
arr = [5, 2, 8, 14]  
target = 10

**Output:**  
false

**Explanation:**  
No combination of four numbers produces the target sum.

---

### Example 4

**Input:**  
arr = [2, 4, 6, 8, 10]  
target = 30

**Output:**  
false

**Explanation:**  
Quadruplet (4, 6, 8, 12) is not possible in this array, but (2, 8, 10, 10) is also invalid due to index reuse.  
If valid distinct elements exist that sum to the target, return true.

---

## Constraints

- 4 ≤ n ≤ 10^5
- -10^9 ≤ arr[i] ≤ 10^9
- -10^9 ≤ target ≤ 10^9
- Array elements may contain duplicates.
- Quadruplets must use distinct indices.

---

## Time Complexity

O(n^2)

---

## Space Complexity

O(n^2)
